## Structure
- **client/**: React frontend (Vite + Tailwind CSS)
- **server/**: Node.js Express backend
- **database/**: Database schemas (MySQL)

## Setup

### Prerequisites
- Node.js installed

### Installation

1. **Backend**
   ```bash
   cd server
   npm install
   node server.js
   ```

2. **Frontend**
   ```bash
   cd client
   npm install
   npm run dev
   ```

## Usage
- Register a new account.
- Login to manage your personal notes.
